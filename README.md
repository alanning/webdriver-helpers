webdriver-helpers
=================

Convenience functions for working with WebDriver in Javascript

<br />

### Usage

```$ npm install webdriver-helpers```

See [online docs](http://alanning.github.io/webdriver-helpers/) for examples.

<br />


### Helpers
* Includes the following functions:
  * populateElements
  * findElementInCollectionByText
  * avoidStaleElement
  * id
  * name
  * tagName
  * className
  * radioByValue
  * selectByValue
  * populateTextBox
  * populateElement


<br />


### Documentation

Online API docs found here: http://alanning.github.io/webdriver-helpers/

To serve documentation locally:
  1. install YUIDoc
  2. ```cd webdriver-helpers```
  3. ```yuidoc --server```
  4. point browser at http://localhost:3000/

To re-generate documentation:
  1. install YUIDoc
  2. ```cd webdriver-helpers```
  3. ```yuidoc```

API docs generated using [YUIDoc][1]
